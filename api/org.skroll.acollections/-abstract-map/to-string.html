toString

open fun toString(): String

Returns a string representation of this map.

The string representation consists of a list of key-value mappings in the order returned by the maps entries views iterator, enclosed in braces ({}). Adjacent mappings are separated by the characters "," (command and space). Each key-value mapping is rendered as the key followed by an equals sign ("=") followed by the associated value.
